# Lauternopoly – Das Kaiserslautern Monopoly

Dieses Repository enthält eine spielbare Browser-Version von **Lauternopoly**, einem Monopoly-inspirierten Spiel mit Straßen und Orten aus Kaiserslautern.

## Live nutzen

1. Repository als ZIP herunterladen und bei GitHub als neues Repository hochladen.
2. In GitHub unter **Settings → Pages** als Source den Branch (z. B. `main`) und den Root-Ordner auswählen.
3. Warten, bis GitHub Pages die Seite gebaut hat.
4. Die Seite ist dann unter der GitHub-Pages-URL spielbar.

## Lokal testen

Einfach die Datei `index.html` im Browser (Chrome, Edge, Firefox, Safari) öffnen.
